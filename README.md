**beatrice** - Simple, minimalist code editor, who works only with keybindings. 

# Feauters 

* All text editor functions. 

* Tabs support. 

* Last files list. 

* Own open file window. 

* Own clipboard window. 

* Commands support 

* Themes support. 

* Simple syntax highlight support. 

* Macros support.

* And something else...
